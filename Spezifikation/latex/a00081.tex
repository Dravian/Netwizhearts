\subsubsection{AusfÃ¼hrliche Beschreibung}
Das \hyperlink{a00081}{Server\-Ruleset} wird im Game\-Server instanziert und verwaltet die Zustände des Game\-States im Server. Mit der Methode \hyperlink{a00081_aa49663b89f89b9f7b3685f0b96560f69}{is\-Valid\-Move()} wird eine Eingabe eines Clients auf Regelkonformität überprüft und dann im Game\-Server das \hyperlink{a00045}{Game\-State} verändert. Über \hyperlink{a00081_afcfbddd7397a6043913b5e742934d2c2}{resolve\-Message()} kann eine Game\-Serverinstanz eine Ruleset\-Message vom Player an das Ruleset weiterleiten.
