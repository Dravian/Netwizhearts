\subsubsection{Ausführliche Beschreibung}
Das Login-\/\-Fenster repraesentiert den initialen Dialog zwischen Benutzer und Client. In diesem Fenster kann der Benutzer seinen Namen und die Adresse des Servers eingeben. Ausserdem ist ueber den \hyperlink{a00017}{Login} die Auswahl der Sprache moeglich. ueber den Login-\/\-Button wird die Verbindung zum Server hergestellt.
